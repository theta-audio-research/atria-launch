ARECIBO

Cinematic Signal-Sculpting Instrument

ARECIBO is a cinematic signal-sculpting instrument built around a layered Source Array. Built around a motion-focused transformation chain, a pre-QUORRA surgical EQ stage, the QUORRA modulation engine, and a final Wall Stage for transient shaping and limiting. Instead of behaving like a generic rack of separate effects, ARECIBO is designed as one continuous transmission system: shape the source, destabilize it, then force it into a controlled boundary. At its core, ARECIBO is about movement. The signal flows through the main engine stages — THETA, DRIFT, BREATHE, FIELD, FADE, and PAN — before entering the more surgical output section. That architecture lets the plugin move from subtle spatial drift and spectral focus to pressure, warp, blur, and controlled collapse, all inside a single insert.

THE SYSTEM

SOURCE ARRAY is the front end of the instrument — the stage where the source behavior is established before it enters the motion chain. In the current interface, Source Array sits as one of the major top-level sections alongside Receiver Chamber, Tweak Array, and Output Surgery, establishing the plugin’s input character before deeper transformation begins. DRIFT introduces temporal instability and motion. It is the first clear step away from static playback, adding movement and looseness before the signal becomes more atmospheric or structural downstream. BREATHE and FIELD continue that process, pushing the signal toward evolving motion and spatial life rather than fixed tone shaping. FADE and PAN complete the core movement chain before the signal reaches the output transformation stages.

INTERFACE

ARECIBO uses THETA Audio Research’s Void design language: dark backgrounds, flat fills, thin vector treatment, restrained metallic accents, and a scientific control-surface aesthetic. The manual defines this visual system around the Void palette, Montserrat for major UI text, JetBrains Mono for values and technical readouts, and a strict no-gradients / no-drop-shadows approach. The visible UI is built around major subsystem cards such as Source Array, Receiver Chamber, Tweak Array, and Output Surgery, with contextual deeper editing revealed only when needed. That keeps the main view focused while still allowing deeper control when the user wants to push into the internals.

QUORRA

QUORRA is the defining transformation stage in ARECIBO. It sits after Output Surgery and before the final wall stage, acting as the last major character engine before limiting. Its purpose is not subtle correction — it is where the sound becomes animated, unstable, and transmitted. The stage was specifically developed around stronger shuffle, doppler-like motion, flange displacement, sweep movement, and node-focused resonance behavior so the signal feels alive rather than simply processed. The current QUORRA control family is built around distinct jobs: Amount for overall intensity, Warp Rate for temporal speed, Warp Depth for displacement severity, Sweep for scan contour, and Node for topology personality.

OUTPUT SURGERY

OUTPUT SURGERY is the pre-QUORRA shaping section. It was specifically reworked to become a fuller, more interactive stage before QUORRA, rather than a simple broad tone stack. The current design direction places a full 8-band EQ here, including HP, LS, Bell 1–4, HS, and LP, so the signal can be frequency-shaped before it is pushed into deeper modulation and final limiting. This section exists because QUORRA responds best when the input has already been sculpted. High-pass and low-pass filtering with resonance/Q, surgical band shaping, and dynamic movement all happen before the signal is destabilized downstream, making the later stages feel more intentional and more dramatic.

WALL STAGE

WALL STAGE is the final output control system. It was simplified so there is only one true final limiter stage, with the EQ remaining fully upstream. In its current structure, Wall Stage combines a small transient design section with the final limiter controls: Attack, Sustain, Input, and Ceiling. This makes the final stage feel more like a mastering wall than simple gain into distortion. The transient section shapes what actually hits the limiter, while Input and Ceiling define the final push and hard boundary before output.
