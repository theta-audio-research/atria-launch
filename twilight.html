TWILIGHT

Regenerative Ambience Engine

TWILIGHT is a living signal environment built for drones, motion, and harmonic atmosphere. At its core is THETA, a four-voice binaural engine built for sustained tone and psychoacoustic interaction. Linked together or driven as a coordinated system, the voices move from meditative tonal beds to immense, architectural harmonic fields. The Signal Lattice turns TWILIGHT from a drone machine into a generative pattern instrument. Editable lanes for gate, pitch, probability, velocity, and swing allow the signal to evolve into pulsing, stepped repetitions and shifting melodic drift. Character is anchored by DRIFT—a harmony-doubler reimagined as a "body-and-halo" architecture. It creates lush, expensive-feeling widening via UNI, OCT, 5TH, MAJ, MIN, SUS, and CLSTR interval families. The Master Output Section provides final-stage treatment: spatial modulation, luxurious stutter behavior, and a true brick-wall limiter designed to act as a deliberate end stage at the edge of the instrument. TWILIGHT is for the space between structure and emergence.

CORE_SYSTEMS

  • Four-Voice THETA Engine Binaural drone system with interval snapping, note behavior, and psychoacoustic motion.
  • Signal Lattice Pattern-driven sequencing with per-voice gate, pitch, probability, velocity, and swing.
  • DRIFT Harmony Doubler Body-and-halo architecture for musical thickening and ensemble-style widening.
  • V5 Sample Layer + THRU Hybrid processing for loaded audio or external tracks routed through the THETA chain.
  • Master Output Matrix Integrated flanger, stutter, and spatial modulation leading to a true brick-wall limiter.

SPECIFICATIONS

  • Platform macOS [AU / VST3]
  • Engine Type Regenerative Ambience / Hybrid Synthesis
  • Direct Capture Real-time output recording to WAV via REC function.
