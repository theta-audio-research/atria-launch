EROSION

Four-Stage Saturation Engine

EROSION treats degradation as a physical process. Built around four distinct analog circuit models, it provides a flexible routing architecture paired with a performance-grade XY control surface. From invisible warmth to aggressive destruction, EROSION covers the full spectrum of nonlinear transformation in a single insert. The system is designed around four hero modules: • Distress: Germanium transistor compressor-saturator with asymmetric clipping. • Amber: Tape saturation topology with tilt EQ for bass warmth or upper-mid air. • Grime: Parallel tube simulator (Pentode, Triode, 12AT7) driven by an XY heat pad. • Grit: Clean tanh waveshaping for symmetrical edge and presence. The visual centerpiece is the GRIME XY pad—a functional surface that simultaneously sets tube character and drives performance-grade effects like Grain Scatter, Stutter Gate, and Feedback Smear. Whether used for subtle mood layering or full signal collapse, EROSION steers sound through a signal path built for technical beauty and controlled failure.

ANALOG_CIRCUITS

  • DISTRESS [GERMANIUM] Biased asymmetric clipping and gain reduction for unpredictable harmonic crunch.
  • AMBER [TAPE] Even-harmonic saturation topology with frequency-dependent tilt weighting.
  • GRIME [TUBE] Parallel Pentode/Triode/12AT7 stages controlled via 2D heat mapping.
  • GRIT [TANH] Symmetrical soft-clipping stage for clean edge and transient definition.

XY_PERFORMANCE

  • Grain Scatter Time-scattered micro-grains with subtle pitch randomization and cloud density control.
  • Stutter Gate Physical drag-velocity rhythm. Signal chops into patterns based on cursor movement.
  • Feedback Smear Resonant comb filtering with controllable pitch and self-oscillation feedback.

ROUTING_INTEL

  • Series Sequential chain (Distress → Amber → Grime → Grit) for compound character.
  • A/B Split Parallel paths (Distress/Amber vs Grime/Grit) for surgical character balancing.
  • Parallel Fully independent summing of all four modules for maximum depth and movement.

TECHNICAL_SPEC

  • Architecture: Stereo 64-bit // Zero Latency (Base)
  • Oversampling: Selectable OFF / 2x / 4x / 8x (Per-Preset)
  • Transients: Post-Saturation Clipper + Tanh Soft Limiter
  • Interface: Composite Concentric Knobs with Gradient Value Arcs
